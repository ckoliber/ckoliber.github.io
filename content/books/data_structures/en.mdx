This book is written on the general topic of data structure, the author in this book introduces the standard data structure and their types with a detailed description. <br />
This book consists of 12 main sections, each of which deals with one of the types of data structures. <br />
The author has written this e-book using the following sources:

-   [CLRS](https://www.amazon.com/Introduction-Algorithms-3rd-MIT-Press/dp/0262033844)
-   [Algorithms](https://www.amazon.com/Algorithms-4th-Robert-Sedgewick/dp/032157351X)
-   [Wikipedia](https://en.wikipedia.org/wiki/Data_structure)

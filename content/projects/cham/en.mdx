---
title: "Chapar Messenger (ChaM)"
date: "2013"
---

Messenger software with the aim of implementing group voice and video calling capabilities, this messenger includes two user interfaces: mobile application (implemented in Java and C languages), desktop application (implemented in C++ and QML languages) and Server application (implemented by Erlang and C languages). <br />
In this application, all communication protocols are based on TCP / UDP Socket and HTTP protocol is not used. Also, Hole Punching method is used to make P2P calls, and ffmpeg library is used to encode / decode audio and video.

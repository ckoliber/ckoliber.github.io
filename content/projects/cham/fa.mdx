### Chapar Messenger (ChaM)

**2013**

نرم‌افزار پیام‌رسان با هدف پیاده سازی قابلیت تماس صوتی و تصویری گروهی، این پیام‌رسان شامل دو رابط کاربری می باشد: اپلیکیشن موبایل (پیاده سازی شده توسط زبان های Java و C)، اپلیکیشن دسکتاپ (پیاده سازی شده توسط زبان های C++ و QML) و اپلیکیشن سرور (پیاده سازی شده توسط زبان های Erlang و C). <br />
در اپلیکیشن مذکور تمامی پروتکل های ارتباطی بر پایه TCP/UDP Socket میباشد و از پروتکل HTTP استفاده نمی‌شود، همچنین جهت برقراری تماس P2P از روش Hole Punching استفاده می‌شود، و جهت Encode/Decode کردن صدا و تصویر از کتابخانه ffmpeg استفاده می‌شود.

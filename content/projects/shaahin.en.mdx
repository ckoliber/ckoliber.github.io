Truck traffic management and bulk freight management system at the port level, this system includes two user interfaces: mobile application (used by port loaders to record traffic), web application (to define basic data, allocation and view statistics and status). <br />
In the mentioned system, all database structures are defined as abstract, and therefore it is possible that the above software can be used to manage the movement of goods in all land, sea and air transport terminals.
